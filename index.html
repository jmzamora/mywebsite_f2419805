Configuración de Git y GitHub

***** Resumen de Comandos *****

1) git clone dirección_de_su_repositorio_git

2) cd nombre_carpeta_de_repositorio

3) git status

4) git add .

5) git commit -m "mensaje de texto"

6) git config --global user.mail "correo_electronico_en_GitHub" (en caso de que salga problema)

7) git push
